View Burst Statistics from all your child sites in one MainWP dashboard. Privacy-friendly analytics, centrally managed.

== Description ==

The Burst Statistics MainWP Extension brings your privacy-friendly analytics into the MainWP dashboard. Monitor visitors, pageviews, and top content across all your child sites without leaving MainWP.

= Requirements =

* [MainWP Dashboard](https://wordpress.org/plugins/mainwp/) on your main site
* [Burst Statistics](https://wordpress.org/plugins/burst-statistics/) (free or Pro) on each child site

= Features =

* Top performing pages and posts per site
* Quick comparison of site performance
* Live visitor counts per site
* Privacy-first: all data stays within your own MainWP and child site infrastructure. Nothing is sent to external servers.

= About Burst Statistics =

Burst Statistics is a privacy-friendly Google Analytics alternative for WordPress, trusted by 200,000+ sites. Built by the team behind UpdraftPlus, WP-Optimize, and All-In-One Security. Learn more at [burst-statistics.com](https://burst-statistics.com).

== Installation ==

1. Install and activate [MainWP Dashboard](https://wordpress.org/plugins/mainwp/) on your main site.
2. Install and activate [Burst Statistics](https://wordpress.org/plugins/burst-statistics/) on each child site you want to track.
3. Install and activate this extension on your MainWP dashboard site.
4. Go to **MainWP > Extensions > Burst Statistics** to view your analytics.

== Frequently Asked Questions ==

= Do I need Burst Statistics on every child site? =
Yes. This extension reads data from Burst Statistics on your child sites. Without it, there is nothing to display.

= Does this work with Burst Pro? =
Yes. Both the free and Pro versions of Burst Statistics are supported. Pro features are available on child sites where Burst Pro is active.

= Where is the data stored? =
All statistics remain on your child sites. The extension only reads and displays them through the MainWP connection; nothing is transmitted to third parties.

= Is an account required? =
No. No external account is needed for this extension itself. A Burst Pro license, if used, is managed on the child site as usual.
